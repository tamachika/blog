文字数をカウントしてくるやつ


                    javascript:(function() {
                        const counterDiv = document.createElement("div");
                        counterDiv.id = "selected-text-counter";
                        counterDiv.style.cssText = "position: fixed; top: 10px; right: 10px; padding: 8px 12px; background: #333; color: white; font-size: 14px; z-index: 9999";
                        document.body.appendChild(counterDiv);
                      
                        const updateCounter = () => {
                          const selectedText = window.getSelection().toString();
                          const textLength = selectedText.length;
                          counterDiv.textContent = `選択した文字列の文字数: ${textLength}`;
                        };
                      
                        document.addEventListener("mouseup", updateCounter);
                        document.addEventListener("keydown", updateCounter);
                      })();

3 名前に任意の名前（例えば、「文字数カウンター」）を入力し、URLに上記のJavaScriptコードを貼り付ける
4 保存

このブックマークレットを使用するには、既存のウェブサイトを開いてから、作成したブックマークレットをクリック。
これにより、選択した文字列の文字数が画面上に表示できます。
